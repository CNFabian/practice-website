# 🎮 Gamified Learning Platform

A simple learning platform with courses and achievements.

## 🚀 Quick Start

1. **Setup:**
```bash
yarn install
cp .env.local.example .env.local
```

2. **Development:**
```bash
yarn dev
```

Visit http://localhost:3000

## 📁 Structure

```
src/
├── app/                 # Pages
├── components/          # UI components
├── types/              # TypeScript types
└── constants/          # Mock data
```

## 🛠️ Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
